# shinyStorePlus R package

In-browser storage for Shiny persistent, synchronized data from the inputs using IndexedDB

## Installation and Usage

### Install

`install.packages(shinyStorePlus)`

### Usage

`library(shinyStorePlus)`

### Use example

`seeexample()`
